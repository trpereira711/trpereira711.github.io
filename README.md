# Consultoria Técnica em Software

Este repositório hospeda meu site pessoal/profissional publicado via GitHub Pages.

## Objetivo do site

Apresentar meus serviços de consultoria técnica para empresas que precisam de apoio especializado em sistemas Java corporativos, incluindo:

- Diagnóstico técnico de aplicações Java/Spring
- Modernização de sistemas legados
- Desenvolvimento e evolução de APIs REST
- Revisão arquitetural
- Code review sênior
- Testes automatizados
- Integrações entre sistemas
- Apoio técnico a equipes de desenvolvimento

## Tecnologias utilizadas

Este site foi pensado inicialmente como uma landing page estática, utilizando:

- HTML
- CSS
- JavaScript
- GitHub Pages

A proposta é manter uma estrutura simples, rápida, leve e fácil de evoluir.
